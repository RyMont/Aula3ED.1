# Exercícios 4 e 5 — Bibliotecas de Ordenação e Projeto de Utilização

Estes exercícios foram reunidos neste mesmo README por tratarem do mesmo contexto de desenvolvimento.

Criar, em Java, uma biblioteca que implemente as funcionalidades de ordenação do BubbleSort e uma biblioteca que implemente as funcionalidades de ordenação do MergeSort.

Em seguida, criar um projeto Java que utilize as bibliotecas criadas e apresente os resultados dos exercícios 1, 2 e 3.
